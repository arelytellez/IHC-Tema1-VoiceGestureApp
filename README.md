# NOVA - Sistema Inteligente de Reconocimiento por Voz y Gestos

## 📌 Descripción del Proyecto

NOVA es una aplicación web interactiva desarrollada como parte del Tema 1: Introducción al desarrollo de IHC.  
El sistema permite el reconocimiento de comandos mediante **voz** y **gestos**, interpretando órdenes específicas en tiempo real.

La aplicación está diseñada bajo principios de Interacción Humano-Computadora (IHC), priorizando usabilidad, claridad visual e interacción intuitiva.

---

## 🎯 Objetivo

Desarrollar una aplicación web capaz de reconocer y devolver específicamente los siguientes comandos:

- avanzar  
- retroceder  
- detener  
- vuelta derecha  
- vuelta izquierda  
- 90° derecha  
- 90° izquierda  
- 360° derecha  
- 360° izquierda  

---

## 🧠 Tecnologías Utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Web Speech API (Reconocimiento de Voz)  
- MediaPipe (Reconocimiento de Gestos)  
- OpenCV (Procesamiento de imagen)  
- GitHub Pages (Despliegue público)

---

## 🖥️ Funcionalidades

### 🎤 Módulo de Voz
- Activación por palabra clave.
- Interpretación de comandos específicos.
- Respuesta en tiempo real.
- Interfaz visual moderna y futurista.

### ✋ Módulo de Gestos
- Detección de movimientos mediante cámara.
- Interpretación de gestos predefinidos.
- Traducción de gesto a comando válido.

---

## 🏗️ Arquitectura del Sistema

El sistema se divide en tres componentes principales:

1. Interfaz Principal (Selección de modo)
2. Módulo de Reconocimiento por Voz
3. Módulo de Reconocimiento por Gestos

Cada módulo opera de forma independiente pero pertenece al mismo entorno de aplicación.
